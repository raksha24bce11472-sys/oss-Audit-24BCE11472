## Getting Started

### Prerequisites

- Bash shell (version 4.0 or higher)
- `curl` and `wget` utilities
- Internet access for fetching package metadata

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/raksha24bce11472-sys/oss-Audit-24BCE11472.git
   cd oss-Audit-24BCE11472
   ```

2. Make scripts executable and run the build:
   ```bash
   chmod +x build script*.sh
   ./build
   ```

## Scripts Overview

| Script      | Purpose                              |
|-------------|--------------------------------------|
| `script1.sh`| Set up the audit environment         |
| `script2.sh`| Audit project dependencies           |
| `script3.sh`| Check open-source licenses           |
| `script4.sh`| Scan for known security issues       |
| `script5.sh`| Generate a consolidated audit report |

---

## Usage

Run each script individually or use the `build` script to execute them in sequence:

```bash
./build
```

Or run a specific script:

```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh
./script5.sh
```
