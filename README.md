## Project Overview
This repository contains five shell scripts created for the Open Source Audit capstone project based on **Git**, an open-source version control system. The scripts demonstrate practical Linux shell scripting concepts such as variables, conditionals, loops, command-line arguments, file handling, and interactive input.

## Repository Contents
- `script1.sh` - System Identity Report
- `script2.sh` - FOSS Package Inspector
- `script3.sh` - Disk and Permission Auditor
- `script4.sh` - Log File Analyzer
- `script5.sh` - Open Source Manifesto Generator
- `manifesto_rave.txt` - Sample output generated by Script 5

## Script Descriptions

### 1. script1.sh - System Identity Report
This script displays important Linux system information such as:
- Linux distribution name
- Kernel version
- Logged-in user
- Home directory
- System uptime
- Current date and time
- A short open-source license message

### 2. script2.sh - FOSS Package Inspector
This script checks whether **Git** is installed on the system. It displays package details and prints a short philosophy note about Git using a `case` statement.

### 3. script3.sh - Disk and Permission Auditor
This script audits important Ubuntu directories such as `/etc`, `/var/log`, `/home`, `/usr/bin`, and `/tmp`. It also checks Git-related files such as:
- `/usr/bin/git`
- `/etc/gitconfig`
- `~/.gitconfig`
- `~/.config/git/config`

### 4. script4.sh - Log File Analyzer
This script reads a log file line by line, searches for a keyword such as `git`, counts the matching lines, and prints the last 5 matching entries. It is designed for Ubuntu package-management logs like:
- `/var/log/dpkg.log`
- `/var/log/apt/history.log`

### 5. script5.sh - Open Source Manifesto Generator
This interactive script asks the user three questions and generates a personalized Git-inspired open source manifesto. The output is saved into a `.txt` file.
